# Simple-portfolio-website

This project is a simple and clean personal portfolio website developed using HTML and CSS. It is designed to showcase my foundational skills in front-end development and to create a basic online presence.

The website currently consists of two main pages: the Home page and the Contact page. The Home page provides a brief introduction and overview, while the Contact page allows visitors to reach out easily. The layout is designed with a focus on simplicity, readability, and user-friendly navigation.

The design follows a minimal approach, ensuring that the content is clear and visually appealing without unnecessary complexity. CSS has been used to style the components, manage layout structure, and maintain consistency across the pages. Basic responsive design techniques have also been applied to ensure the website adapts well to different screen sizes such as desktops, tablets, and mobile devices.

The website includes two main pages:

🏠 Home Page – Provides a brief introduction and overview
📩 Contact Page – Allows users to get in touch easily

The design is minimal, responsive, and user-friendly, ensuring a smooth experience across different devices.
